# Sakura

用于看番的app，之后也支持看漫画和小说。

目前看番存在的问题:

- 某些稳定的视频源收费，并且和谐严重
- 国内浏览器屏蔽了看番网站
- 国外浏览器看番体验感差
- 某些app数据源不稳定，容易挂
- 某些app UI设计较为简单，不够人性化
- 某些开源的app不提供安装包，需要自己build，用户体验感差

## 特点：
- 视频数据和搜索功能由[www.yinghuacd.com](http://www.yinghuacd.com/)提供
- react native实现
- 具体的UI设计参考了B站
